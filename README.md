# 🐺 AK Pack Fitness — Attendance System

A lightweight gym attendance and membership monitoring system built for **GitHub Pages + Google Sheets**.

## Features

- ✅ **Check-In Terminal** — Members enter their ID at reception
- ✅ **Membership Status** — Active / Warning / Expired with color indicators
- ✅ **Renewal Alerts** — Automatic alerts at 3 days, 1 day, and expired
- ✅ **Live Admin Monitor** — Real-time attendance dashboard, auto-refreshes every 8s
- ✅ **Pack Counter** — Shows how many members are in the gym today
- ✅ **Attendance History** — Full log with date, ID, and name filters
- ✅ **Google Sheets Database** — Free, no backend required
- ✅ **GitHub Pages Hosted** — Free, fully online

---

## Quick Start

1. **Clone or download** this repository
2. Follow the **[SETUP_GUIDE.md](SETUP_GUIDE.md)** to connect Google Sheets
3. Upload to GitHub and enable GitHub Pages
4. Done! 🎉

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript (vanilla)
- **Database:** Google Sheets (via public JSON API + Apps Script)
- **Hosting:** GitHub Pages (free static hosting)

---

*No backend. No server costs. Just a browser and Google Sheets.*
